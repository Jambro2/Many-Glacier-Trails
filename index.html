Many Glacier Valley

Created by Jamie

Full-screen version

Many Glacier Valley has some of the best hiking and scenic views in Glacier National Park, Montana. If you are taking a once in a lifetime trip to GNP, then you have to visit the Many Glacier Valley. The best trail, in my opinion, is the Grinnell Glacier Trail. This map also includes amenties like campgrounds, bathrooms (man/woman symbols), and parking. The camera icons are scenic viewpoints. If I was QGIS savvy, I would link pics to these viewpts. I would also like to add layers showing the effects climate change has had on Grinnell Glacier. In 1850, there were 80 glaciers in the park and now only 26 remain. According to USGS data, Grinnell has lost 100 acres in size from 1966 to 2015.

Photograph of Grinnell Glacier and Upper Grinnell Lake in 2018 taken by Jambro2

To create this map, the main tool used was QGIS QuickOSM Plugin and OpenStreetMap WIKI.
  • Data for the elevation contours accessed from USGS.
  • Coordinate Reference System EPSG: 32100 NAD83/Montana
  • XYZ Tiles(Directory) processing tool to export the map with a Zoom Min 10 and Zoom Max 15.

    • Created on QGIS version #3.32.2 using OpenStreetMap data harvested from the QuickOSM plugin. © 2022 Openstreet map contributors.
